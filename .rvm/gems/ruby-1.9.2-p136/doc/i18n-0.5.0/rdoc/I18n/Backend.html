Constants

USE_INSPECT_HASH

In Ruby < 1.9 the following is true: { :foo => 1, :bar => 2 }.hash == { :foo => 2, :bar => 1 }.hash Therefore we must use the hash of the inspect string instead to avoid cache key colisions.
